# Neural Networks from Scratch

## Kebutuhan Sistem

- Python 3.10 atau lebih baru
- [pip](https://pip.pypa.io/en/stable/)
- [uv](https://github.com/astral-sh/uv) (opsional,tapi lebih cepat)

## Instalasi Dependencies

### Menggunakan pip

```bash
python -m venv .venv
source .venv/bin/activate # Di Windows pakai `.venv\Scripts\activate`
pip install -r requirements.txt
```

### Menggunakan uv

```bash
uv venv
source .venv/bin/activate # Di Windows pakai `.venv\Scripts\activate`
uv sync
```
